# 🥗 Diet Recommendation Chatbot App

An AI-powered web application that generates personalized diet plans based on user profiles, health inputs, preferences, and fitness goals. The system integrates a conversational chatbot to guide users, answer dietary queries, and provide smart nutritional recommendations.

🚀 Features
🧠 AI-Powered Diet Planning
Generates customized meal plans based on user inputs like goals, preferences, and health data
💬 Interactive Chatbot
Uses AI to assist users, answer diet-related questions, and provide guidance
⚙️ Automated Meal Planning Workflow
Streamlined system for generating structured diet plans
🍽️ Nutrition Data Integration
Fetches food and nutrition details using Spoonacular API
🌐 Responsive Web Interface
User-friendly frontend for smooth interaction

### 🛠️ Tech Stack

Frontend:
HTML
CSS
JavaScript

Backend:
Python
Flask

APIs & Tools:
Google Generative AI (for chatbot intelligence)
Spoonacular API (for food & nutrition data)

📌 How It Works
User enters details (age, goals, preferences, etc.)
Backend processes inputs using logic + APIs
AI chatbot interacts with the user conversationally
System generates a personalized meal plan
Nutritional insights are displayed dynamically
